This is a WordPress theme

Theme resources:
Built with Bootstrap 3
Valid HTML5/CSS3 Markup
jQuery v1.11.1
Google Fonts Support
Clean Code, All files are well commented and organized
Included WOW animations
Font Awesome 4.3 and Simple Line Icon support

Simplified administration in theme options

Installation:
Create a clean folder in WordPress themes folder and place all files and directories.
Activate clean theme.

Bootstrap 3 
HTML5/CSS3

Ative o tema clean.
